Calculadora de Rank de Herói

Este projeto foi desenvolvido com o objetivo de praticar conceitos básicos de lógica de programação utilizando JavaScript.

O programa calcula o saldo de vitórias de um herói a partir da quantidade de vitórias e derrotas informadas pelo usuário e, com base nesse saldo, define o nível do herói.

📌 Objetivo do Projeto

Criar um algoritmo que:

Receba do usuário o número de vitórias.

Receba o número de derrotas.

Calcule o saldo de vitórias.

Classifique o herói em um nível baseado nesse saldo.

Mostre o resultado na tela.

⚙️ Tecnologias Utilizadas

JavaScript

Lógica de programação

Execução em navegador (usando prompt, alert e console.log)
